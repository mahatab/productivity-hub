# M2LABS Website Sample Home Pages

Three different design directions for the m2labs website.

## 📁 Files

### Option 1: Brutalist/Neo-Minimal (option1-brutalist.html)
**Style:** Black & white, bold typography, high contrast
**Vibe:** Raw, honest, no-nonsense
**Inspiration:** Linear.app, brutalist design movement
**Best for:** Making a strong statement, tech-forward audience

**Features:**
- Pure black background with white text
- Massive typography (120px headlines)
- Hard borders and geometric layout
- Hover effects that invert colors
- Very bold and uncompromising

---

### Option 2: Clean & Modern (option2-clean.html)
**Style:** Soft gradients, rounded corners, polished
**Vibe:** Professional, friendly, approachable
**Inspiration:** Stripe, Superhuman
**Best for:** Broader audience appeal, mainstream users

**Features:**
- Purple gradient hero section
- Soft shadows and rounded corners
- Icon-based feature cards
- Light background (#fafafa)
- Smooth hover animations
- Very polished and professional

---

### Option 3: Bold & Colorful (option3-bold.html)
**Style:** Strong accents, grid layout, energetic
**Vibe:** Confident, dynamic, startup energy
**Inspiration:** Modern SaaS products
**Best for:** Standing out, showing personality

**Features:**
- Orange (#FF6B35) accent color
- Numbered principles (01, 02, 03, 04)
- Navigation menu in header
- Two-column grid layout
- Black project highlight section
- High energy, very confident tone

---

## 🎯 Recommendations

**If you want maximum impact and edge:** → Option 1 (Brutalist)
**If you want professional polish:** → Option 2 (Clean)
**If you want bold personality:** → Option 3 (Bold)

## 🚀 How to View

1. Open any HTML file in your browser
2. Each is fully responsive (works on mobile)
3. All styles are inline (no external dependencies)
4. Ready to customize

## ✏️ Customization Notes

All three samples use:
- System fonts (works everywhere, loads instantly)
- Pure CSS (no JavaScript needed for now)
- Semantic HTML (SEO-friendly)
- Mobile-responsive design

**Easy changes:**
- Colors: Search for color codes (e.g., `#FF6B35`) and replace
- Text: Just edit the HTML content
- Fonts: Change the `font-family` in CSS
- Layout: Adjust `max-width` values in `.container`
