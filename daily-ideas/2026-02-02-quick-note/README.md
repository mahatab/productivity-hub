# Quick Note - Minimal Note Taking App

**Built:** February 2, 2026  
**Idea:** A distraction-free note-taking web app that auto-saves to browser localStorage

## Why This Exists

Sometimes you just need to jot something down. No account. No cloud sync. No features you don't need. Just a clean textarea that saves your thoughts automatically.

## Features

- ✅ **Auto-save** - Saves to localStorage every second after you stop typing
- ✅ **Character count** - Track your note length
- ✅ **Download** - Export as .txt file
- ✅ **Copy to clipboard** - One-click copy
- ✅ **Clear note** - With confirmation to prevent accidents
- ✅ **Beautiful design** - Gradient background, clean interface
- ✅ **No dependencies** - Pure HTML/CSS/JS
- ✅ **Works offline** - Fully client-side

## Use Cases

- Quick thoughts and ideas
- Temporary note-taking
- Clipboard manager
- Draft before pasting elsewhere
- Privacy-conscious note-taking (nothing leaves your browser)

## m2labs Insight

People want simple tools that do one thing well. NoteTabs (minimal text editor) launched a month ago and got traction with ZERO AI features. This proves: **Simple + Fast > Complex + Perfect**.

Quick Note is even simpler - no install, no files, just open and type.

## Tech Stack

- Pure vanilla JavaScript
- CSS3 (gradients, animations)
- localStorage API
- Clipboard API

**File:** `index.html` (single file, ~6KB)
