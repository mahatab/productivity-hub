# Focus Timer - Example Daily Idea

**Date:** January 30, 2026  
**Inspired by:** News overload / need for focused reading time

## The Problem

People start reading news articles but get distracted by notifications, side links, and endless scrolling. They never finish what they started.

## The Solution

**Focus Timer** - A simple countdown timer that:
- Estimates reading time based on article length
- Full-screen distraction-free mode
- No pause button (commitment device)
- Gentle chime when done
- Track completion streak

## m2labs Fit

✅ **Single purpose:** Just helps you finish reading  
✅ **Simple:** One button to start, no settings overload  
✅ **Respects time:** No infinite scrolling or dark patterns  

## Prototype

Built in `index.html` - working demo with:
- Clean minimal UI
- Estimated reading time calculation
- Full-screen mode
- Completion tracking
- No distractions
